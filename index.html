WebGet by luigimax

WebGet is a html parser and selective mirror tool. Will be extensible with modules to do selective mirroring of other sites

Dependencies

Maven 2. it will handle the rest of the dependancies

Install

1)Get git 2)Get Maven 2 3)pull from project public repo 4)in console type :mvn scala:run to run
